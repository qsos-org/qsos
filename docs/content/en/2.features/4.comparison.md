**Comparison** is the final step of the QSOS methodology. It allows you to put evaluated software side by side, apply your weightings and visualize the results. This is **Phase 4** of the methodology.

---

## STEP 1: Select Software

From a software type, select the software and versions you want to compare, along with the evaluation grid version.

## STEP 2: Comparison Table and Weights

The table displays software side by side with scores for each criterion, organized by sections and subsections. You can load a requirements preset or adjust weights manually — scores are recalculated in real time.

## STEP 3: Visualize and Export

Two chart types are available:

- **Radar**: overview of scores by criterion
- **Bubble**: software positioning along two axes of your choice

## Best Practices

- Compare software of the same type only
- Use presets for reproducible comparisons
- Document the comparison context
- Involve multiple evaluators for objectivity
