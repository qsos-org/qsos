---
title: Préréglages d'exigences
description: Créer et gérer les préréglages de pondération pour les comparaisons QSOS
---

Les **préréglages d'exigences** vous permettent de sauvegarder et réutiliser des **pondérations personnalisées** lors de la comparaison de logiciels. C'est la **Phase 3** de la méthodologie.

Un préréglage définit l'importance relative de chaque critère d'évaluation selon les priorités de votre organisation (par exemple, la sécurité prime sur la performance dans un contexte bancaire).

---

## ÉTAPE 1: Créer un préréglage

Créez un préréglage depuis la **vue des préréglages** ou directement depuis une **page de comparaison**.

::doc-image
---
src: /images/presets/create-preset-fr.png
alt: Création d'un préréglage
caption: Figure 1 — Création d'un préréglage
maxWidth: 350px
---
::

Remplissez un **nom** clair et optionnellement une **description** pour spécifier le contexte d'utilisation (par exemple, "Priorité sécurité - secteur bancaire"). Le préréglage est lié à une version de **grille** et un **type de logiciel** spécifique.

<details>
<summary>Voir le JSON</summary>

```json
{
  "presetUid": "preset-001",
  "label": "Priorité sécurité",
  "description": "Pondération orientée sécurité pour le secteur bancaire",
  "userUid": "alice@company.fr",
  "gridVersion": "1.0",
  "softwareTypeUid": "web_frameworks",
  "criteriaWeights": {
    "sections": [
      {
        "weight": 800,
        "sections": [
          { "weight": 700 },
          { "weight": 900 }
        ]
      },
      {
        "weight": 500,
        "sections": [
          { "weight": 400 }
        ]
      }
    ]
  },
  "createdAt": "2025-09-08T14:00:00.000Z",
  "updatedAt": "2025-09-08T14:00:00.000Z"
}
```

</details>

## ÉTAPE 3: Gérer vos préréglages d'exigences

Trouvez tous vos préréglages d'exigences depuis votre **profil** → **"Mes Préréglages d'Exigences"**. Vous pouvez consulter, modifier ou supprimer vos préréglages existants.

## Bonnes pratiques

- Nommer vos préréglages explicitement (contexte + priorité)
- Créer différents préréglages pour différents contextes métier
- Réutiliser les préréglages pour des comparaisons cohérentes dans le temps
- Mettre à jour vos préréglages lorsque la grille évolue
